# 404 Page — Terminal-Style Not Found

## Overview

Revamp the root 404 page to use the frontend layout (Header, Footer, MatrixBackground, ThemeProvider) and display a friendly terminal-themed "not found" experience with a typing animation and navigation links.

## Architecture

The file is `src/app/not-found.tsx`. In Next.js App Router, the root `not-found.tsx` renders inside the root layout (`src/app/layout.tsx`), which is a passthrough (`return children`). The `(frontend)` route group layout has all the styling (HTML/body, fonts, ThemeProvider, Header, Footer, MatrixBackground).

**Approach:** The root `not-found.tsx` will import and compose the frontend layout components directly, providing its own full HTML structure. This avoids restructuring layouts.

Components to import:
- `ThemeProvider`
- `Header`
- `Footer`
- `MatrixBackground`
- Fonts: `JetBrains_Mono`, `Space_Grotesk` from `next/font/google`

The page itself will be a client component (for the typing animation), wrapped in the server-rendered layout shell.

**File structure:**
- `src/app/not-found.tsx` — layout shell (server component), renders `<NotFoundContent />`
- `src/components/NotFoundContent.tsx` — client component with the terminal UI + typing animation

## Terminal UI

### Visual Design

A centered card styled as a terminal window:
- **Title bar:** Three dots (red/yellow/green circles) + label "tech_atlas"
- **Body:** `bg-card` background, `font-mono`, padded content area
- **Border:** `border-border`, `rounded-lg`, subtle shadow

### Content — Typing Sequence

Uses the site's `>` prompt character (matching the header logo). Three lines type out sequentially:

```
> buscando página...
> no encontramos lo que buscas
> te puede interesar:
```

After the last line finishes typing, navigation links fade in.

### Navigation Links

A 2x2 grid of links to main sections, using Lucide icons already in the project:

| Link | Icon | Href |
|------|------|------|
| Directorio | `FolderOpen` | `/directorio` |
| Eventos | `CalendarDays` | `/eventos` |
| Noticias | `Newspaper` | `/noticias` |
| Empleos | `Briefcase` | `/empleos` |

Each link styled as: icon + label, `font-mono text-sm`, `text-secondary hover:text-accent` transition.

Below the terminal card: a "Volver al inicio" link pointing to `/`.

### Animation Details

- Typing speed: ~40ms per character
- Pause between lines: ~400ms
- Blinking cursor (`▌`) follows typing position, blinks via CSS animation
- After all lines complete, links fade in with a CSS transition (`opacity 0→1`, ~300ms)
- Implementation: `useState` + `useEffect` with `setTimeout` chains. No external libraries.

## Responsive Behavior

- Terminal card: `max-w-2xl w-full`, horizontal padding scales with breakpoints
- Link grid: 2 columns on all sizes (4 items, 2x2 is clean everywhere)
- Vertically centered in the main content area using flexbox

## Theme Support

Inherits all theme variables from ThemeProvider — works in both light and dark mode automatically since it uses the existing design tokens (`bg-card`, `text-primary`, `text-secondary`, `text-accent`, `border-border`, etc.).
